# Citrate Buffers

## Overview

DESCRIPTION:

Sodium citrate-citric acid buffer is an acidic bufering medium in the range of pH 3-6.2 used for biochemistry and and molecular biology. Widely used in RNA isolation as it prevents base hydrolysis. For the same reason saline Sodium Citrate (SSC) buffer is widely used in in-situ hybridisation reactions. For immunohistochemistry it is said to increace signals without increacing background.

## Citrate Buffer

PREPARATION

Stock Solutions:                                                                           1 liter

Solution A(0.2M Citric Acid)  1 liter

Trisodium Citrate (Na3C6H507*2H20)                         (MW = 210.14)                      42.8 gm

+ ddH20 to make  1 liter

Solution B (0.2M Trisodium Citrate)  1 liter

Trisodium Citrate (Na3C6H5077*2H20)                           (MW = 294.12)                      42.8 gm

+ ddH20 to make  1 liter

1N NaOH     100 ml

NaOH                                   (MW = 40)                        4 gm

+ ddH20 to make    100ml

1N HCl Solution     100 ml:

8.3 ml fuming HCl (37% v/v = ~12N = ~12M)

91.7 ml ddH20 Store at room temperature for 3 mo or at 4°C for 6 mo.                                                                100 ml

Working buffer:  0.2M                    100 ml

Mix Solution A (Citric Acid) and Solution B (Trisodium citrate) in the given ratio. Set pH using 1N HCl and IN NaOH if needed.

## Saline Sodium Citrate Buffer

DESCRIPTION

20X SSC buffer is a buffer 3M for NaCl and 0.3 M with respect to  Sodium Citrate at pH=7.0 .

PREPARATION

Components:

NaCl (MW: 58.4 g/mol)

Sodium Citrate (mw: 258 g/mol)

Stock Solution:

1. Prepare 800 mL of dH2O in a suitable container.
2. Add 175.3 g of NaCl to the solution.
3. Add 77.4 g of Sodium Citrate to the solution.
4. Adjust solution to desired pH with ccHCl.
5. Add ddH20 until volume is 1 L.

Working Buffer: 1x
