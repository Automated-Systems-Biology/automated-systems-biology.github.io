# Tris Buffers

## Overview

DESCRIPTION:

Tris buffers can be used in a biologically relevant pH range of pH=7.1-8.9. It is an organic compound with a zwitterionic structure that intercalates metal ions and inactivates many enzymes (but is still regarded as physiologically inert).  widely used in chemistry and molecular biology especially in agarose and SDS page agarose gel electrophoresis (TBA, TAE etc). It has a good buffering capacity at higher pH required for some tissues. It is important that the buffer pH is temperature dependent.

## TRIS Buffer

PREPARATION:

Components:

Tris Base/Trisma Base ((HOCH2)3CNH2 ,MW: 121.14 g/mol)

Stock Solutions:

1M TRIS Buffer 1l

1. Prepare 800 mL of distilled water in a suitable container.
2. Add 121.14 g of Tris base to the solution.
3. Adjust solution to desired pH using HCl (typically pH ≈ 7.0).
4. Add ddH20until volume is 1 L.

0.2M TRIS Buffer 100ml

1. Prepare 50 mL of distilled water in a suitable container.
2. Add 24.2 g of Tris base to the solution.
3. Adjust solution to desired pH using HCl
4. Add ddH20 until volume is 100 ml.

Working buffer:  0.1M                         100 ml

1. For example using 0.2M Tris Stock solution start with 50ml Stock.
2. Set pH using 0.1N NaOH .
3. Add ddH20 until 100ml
